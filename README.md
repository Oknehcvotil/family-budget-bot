# family-budget-bot

Простой Telegram-бот для тестового управления семейным бюджетом.

## Установка

1. Установите Node.js 18+.
2. Откройте папку проекта и выполните:

```bash
npm install
```

3. Создайте `.env` (в корне) и вставьте:

```env
BOT_TOKEN=ваш_токен
ALLOWED_USER_IDS=123456789,987654321
```

- `ALLOWED_USER_IDS` — список Telegram user_id владельцев бота, через запятую.

4. Запустите бота:

```bash
node index.js
```

## Команды

- `/start` - приветствие
- `/help` - справка
- `/add` - добавить расход
- `/report` - отчет (пока заготовка)

## Как тестировать

В Telegram: найдите бота по username, отправьте `/start`, затем `150 такси`.
